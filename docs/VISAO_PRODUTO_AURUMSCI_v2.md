## Duas Portas de Entrada

### Porta A - Personal envia link
- Personal cadastra no PRO: nome, email, telefone, objetivo, nivel, sexo, CPF, cobranca
- Sistema gera link unico: aurumsc.com.br/onboarding/TOKEN
- Personal envia via WhatsApp (botao ja funciona)
- Aluno preenche questionario - recebe treino base
- App ja vem com tudo

### Porta B - Aluno auto-cadastro pagante
- Aluno entra em aurumsc.com.br (landing)
- Ve plano, clica QUERO TREINAR
- PAGA pelo Stripe
- Recebe MESMO link de questionario
- Preenche - cria conta - tem treino
- WOW factor: "fiz um bom negocio, ja tenho treino!"

Os DOIS caminhos salvam na MESMA fonte de dados.

## Postural - Solucao Genial do Andre

### Problema
- Aluno nunca fez analise postural
- Nao sabe distancia, angulo
- Foto sai borrada
- Sem alguem pra ajudar
- Desiste

### Solucao
- Vídeo unico de 15 segundos
- 5s parado em cada posição (frente, lado, costas)
- Aluno pausa o video e tira PRINT de cada pose
- Joga os 3 prints no app
- Auri orienta tudo passo a passo

### Por que é genial
- Aluno sozinho consegue (tripe/parede)
- Posicionamento controlado (5s parado)
- Qualidade boa (vídeo permite escolher melhor frame)
- Sem stress, sem vergonha de pedir ajuda

## Reavaliacao Automatica

A cada 8-12 semanas (postural nao muda toda hora).
Sistema dispara link automatico via WhatsApp + Email.
Aluno preenche nova rodada.
Treino atualiza.
Ciclo se repete.
